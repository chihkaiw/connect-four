# CONNECT FOUR

## Run the project locally
1.Clone the project and go to the project folder
2.Create jar file
```sh
gradle clean build
```
3.Run Application
```sh
java -jar build/libs/connect-four.jar
```
4.Find GAME START message on the STDOUT.
```sh
GAME START!!
| | | | | | | |
| | | | | | | |
| | | | | | | |
| | | | | | | |
| | | | | | | |
| | | | | | | |
```
5.Enjoy the match.
